Uplink Developer CD

 

Welcome to the Uplink Developer CD. On this disk you will find everything you need to get started making your own patches for Uplink. We've included all the source code, our own libraries, design docs, and tools that we used to produce the game itself. Everything you need is here.

Feel free to jump straight in to the source code and get started. Just open up the project file in Developer Studio, and have a poke around. We would recommend you at least read the Getting Started guide, and the License, as both contain important information.

Have fun!

Chris
